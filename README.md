## 🏞️ Overview  

Welcome to my **interactive photography portfolio** — a visual playground where creativity meets technology.  
This site blends **modern web design** and **aesthetic visuals** to showcase my photography, artistic style, and frontend skills in an immersive, scroll-triggered experience.

## 🌟 Features  

| Feature | Description |
|----------|-------------|
| 🎞️ **Fade Image Slider** | Smooth, cinematic transitions between hero images |
| 🖼️ **Thumbnail Navigation** | Quick and elegant image selection |
| 💬 **Image Captions** | Contextual overlays to tell each photo’s story |
| 🎯 **Scroll Animations** | Motion effects for an engaging browsing flow |
| 🌙 **Dark/Light Mode Toggle** | Seamless theme switching for user preference |
| 🟡 **Yellow-Themed Fonts** | Distinctive visual identity and warmth |
| 🔍 **Lightbox View** | Zoom into every detail of your best shots |
| 💤 **Lazy Loading** | Fast and performance-optimized gallery |
| 📱 **Responsive Design** | Perfect across mobile, tablet, and desktop |
| 🍔 **Hamburger Menu** | Sleek navigation for smaller screens |
